## Disclaimer
This tool is **not** intended for gaining an unfair advantage or exploiting the game. While it may be possible to misuse it, doing so is against the rules and can result in in-game moderation or bans. Custom tweaks are at your own risk. All provided presets adhere to the terms of service and will not result in punishment. Do not modify enemy players.

A guide on obtaining relevant hashes will not be shared due to its potential for abuse.

If you want a texture added, ping either **@cro.p**, **@8ar**, or **@daokoshoegaze** on our [Discord server](https://discord.gg/v9gXTuCz8B).

## Getting Started
- Run `run.bat` to start the program. The script will guide you through the necessary steps.
- In-game texture replacements provide essential presets for your needs. Each preset is self-explanatory.
- Make sure to load `delta`, `coyote`, `reflex`, `eotech`, or any relevant optic before running commands, or they may not be found.
- The `Custom` option is for items not natively provided but available in the hash dump or if you have your own.
- `Default skyboxes` replace all skyboxes on every map with the default one, allowing modification via Bloxstrap for custom reflective gun skins or unique skyboxes.

## Experimental Features
- **Block**: Highly experimental. Requires running `run.bat` as admin to access. It is volatile and can break your game. Use only if you know what you're doing. It is expected that 99% of users will avoid it.

- **Clear Cache**: Use this if a clear undo option is apparent or if critical bugs occur. It will reset everything you have done.

- **Change Config**: For adding or sharing natively provided options internally.
